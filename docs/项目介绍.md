# OpenStar XFDashboard - 个人财务管理系统

## 项目简介

**OpenStar XFDashboard** 是一款功能完善的**开源个人财务管理 Dashboard**，致力于帮助用户轻松管理个人资产、追踪消费支出、规划储蓄目标和管理贷款债务。

- **当前版本**: 2.1.8
- **发布日期**: 2026-03-19
- **GitHub**: https://github.com/xingseven/openstar-xfdashborad
- **技术架构**: 前后端分离
- **许可证**: 开源免费

---

## 核心功能概览

| 模块 | 功能亮点 |
|------|----------|
| **Dashboard 首页** | 净资产聚合、收支统计、预算预警、快捷入口 |
| **资产管理** | 多币种支持、账户类型丰富、自动估值 |
| **消费分析** | 12+ 图表类型、桑基图、AI 智能记账 |
| **储蓄目标** | 进度追踪、打卡计划、存取记录 |
| **贷款管理** | 还款进度、还款计划、平台分布 |
| **预算管理** | 三维预算（全局/分类/平台）、预警机制 |
| **AI 智能记账** | 拍照识别、三平台支持、自动分类 |
| **连接管理** | 设备授权、连接码、安全管控 |

## 功能详解

### 1. Dashboard 首页 (`/`)

首页汇聚了用户最关心的财务数据：

- **净资产/总资产/总负债** 聚合卡片展示
- **本月收支** 统计卡片（含趋势指示器）
- **最近交易** 列表预览
- **预算预警** 横幅（显示预警/超支状态）
- **快捷入口** 导航按钮

### 2. 资产管理 (`/assets`)

全面管理您的各类资产：

- **账户类型**: 现金、银行卡、支付宝、微信钱包、投资账户、信用卡等
- **多币种支持**: CNY, USD, EUR, HKD, JPY, GBP
- **自动估值**: 多币种资产自动换算为人民币总估值
- **CRUD 操作**: 新增、编辑、删除资产
- **MockData 降级**: 无数据时展示示例数据

### 3. 消费分析 (`/consumption`) - 核心模块

提供详尽的消费数据可视化分析：

#### 数据可视化图表（12 种）

| 图表类型 | 说明 |
|----------|------|
| **支出趋势图** | 折线图，支持按日/按月切换 |
| **消费分类堆积图** | 柱状图，Top 5/10/20 分类 |
| **平台分布饼图** | 环形图，支付宝/微信占比 |
| **收支对比环形图** | 收支结构分析 |
| **热门商家柱状图** | 条形图，Top 10 商家 |
| **每日平均消费** | 日历网格图，周数选择 |
| **资金流向桑基图** | 4 级分支：收入来源 → 支付账户 → 支出去向 |
| **消费时段散点图** | 24 小时分布，发现消费习惯 |
| **单笔金额直方图** | 金额区间分布 |
| **工作日 vs 周末** | 周一至周日统计对比 |
| **分类帕累托图** | 累计占比组合图 |
| **消费热力图** | 平台 × 分类矩阵 |

#### 交互功能

- **智能筛选栏**: 吸顶效果，支持搜索、平台筛选、月份切换
- **AI 拍照记账**: 上传小票，AI 自动识别金额/商户/日期/分类
- **交易流水**: 分页展示，支持编辑/删除
- **CSV 导出**: 支持按筛选条件导出

### 4. 储蓄目标 (`/savings`)

帮助您实现储蓄目标：

- **目标卡片**: 显示进度条、当前/目标金额
- **储蓄类型**: 每月存、隔月存（单/双）、长期储蓄
- **存款类型**: 现金、死期、他人帮存
- **打卡计划**: 分月存款计划，支持"指定计划"功能
- **取款功能**: 从储蓄目标提取资金
- **存取记录**: 自动创建交易记录
- **批量操作**: 批量归档/删除

### 5. 贷款管理 (`/loans`)

清晰追踪贷款和还款：

- **贷款列表**: 平台、总金额、剩余金额、还款进度
- **还款进度**: 环形图可视化
- **还款计划**: 自动推算未来还款日程
- **还款登记**: 记录实际还款
- **平台分布饼图**: 各平台贷款占比
- **已还 vs 剩余柱状图**: 对比展示

### 6. 预算管理 (`/budgets`)

多维度预算控制，避免超支：

- **三种预算作用域**:
  - `GLOBAL`: 全局预算
  - `CATEGORY`: 分类预算
  - `PLATFORM`: 平台预算
- **自定义预警阈值**: 默认 80%
- **健康状态指示**:
  - 绿色: 正常（< 80%）
  - 黄色: 预警（80% - 100%）
  - 红色: 超支（> 100%）
- **周期选择**: 月度/年度

### 7. AI 智能记账 (`/ai`)

#### 大模型配置管理

- **多模型支持**: 火山引擎(豆包)、OpenAI、Anthropic、自定义
- **视觉模型**: 支持拍照记账
- **设为默认**: 优先使用默认模型
- **配置管理**: API Key 安全存储
- **连接测试**: 验证配置有效性

#### AI 拍照记账功能

**支持三大支付平台**：

| 平台 | 识别字段 |
|------|----------|
| **支付宝** | 商户名称、消费日期、账单分类、付款方式、收款方全称、支付时间、备注 |
| **微信支付** | 商户名称、商品描述、支付时间、商户全称 |
| **云闪付** | 消费名称、卡号（尾号）、交易时间（精确到秒）、交易类别 |

- **隐私保护**: 图片 Base64 处理，不占用服务器磁盘
- **用户确认机制**: 识别结果可微调后确认

### 8. 连接管理 (`/connections`)

多设备安全管理：

- **连接码生成**: OTP 6 位码，5 分钟有效期
- **设备列表**: 查看已授权设备
- **撤销授权**: 吊销已连接设备

### 9. 设置 (`/settings`)

- **个人信息**: 修改昵称
- **密码修改**: 旧密码 + 新密码

### 10. 关于 (`/about`)

- **项目介绍**: 版本、技术栈、主要功能
- **版本历史**: 展开/收起式列表
- **贡献者**: GitHub Contributors
- **相关链接**: GitHub 仓库/Issue/Discussions

## 特色功能

### 1. 智能消费分析

- **资金流向桑基图**: 直观展示资金从收入来源到支付账户再到支出去向的全流程
- **消费时段分析**: 24 小时散点图，发现您的消费习惯
- **金额分布直方图**: 识别"细水长流"或"大额低频"消费模式
- **NLP 智能分类**: 100+ 关键词规则，自动识别商家分类

### 2. 异常消费检测

- **大额支出检测**: 默认阈值 ¥1000
- **高频消费检测**: 30 天内超过 10 次
- **新类别检测**: 发现新出现的消费模式
- **风险分级**: LOW / MEDIUM / HIGH 三级

### 3. 事件总线架构

支持异步事件处理：

- `transaction:import:start / complete / error`
- `transaction:create / delete`
- `budget:alert`
- `anomaly:detected`

### 4. PWA 离线支持

- Service Worker 注册
- 离线访问
- 安装到主屏幕
- 离线提示页面

### 5. 移动端适配

- Hamburger 菜单
- 侧滑抽屉导航
- 图表横向滚动
- 响应式布局

### 6. 性能优化

- **ECharts 替代 Recharts**: 解决 SVG 万级节点卡顿
- **next/dynamic 异步加载**: 页面切换零等待
- **IntersectionObserver**: 图表懒加载
- **DelayedRender**: 分批串行加载，渐次入场动画
- **骨架屏**: 像素级高度匹配，消除布局跳动

## 快速开始

### 安装依赖

```bash
# 安装前后端依赖
npm install

# 初始化数据库
cd src/server
npx prisma migrate dev
```

### 启动开发服务器

```bash
# 前后端同时启动
npm run dev
```

- 前端: http://localhost:3000
- 后端: http://localhost:3004
