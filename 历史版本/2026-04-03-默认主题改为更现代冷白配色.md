# 2026-04-03 默认主题改为更现代冷白配色

## 本次变更

- 将默认主题从偏暖白、偏土黄米棕的复古感配色调整为冷白、浅雾灰、靛蓝点缀的现代浅色方案。
- 保留简约结构，但把首页卡片、图表、导航和交易列表的强调色改得更年轻、更清爽。
- 同步移除上一版中容易显旧的暖土色倾向，避免界面出现“复古”“旧后台”的感觉。

## 涉及文件

- `web/src/themes/registry.ts`
- `web/src/app/globals.css`
- `web/src/components/shared/theme-primitives.tsx`
- `web/src/features/dashboard/components/themes/DefaultDashboard.tsx`
- `CHANGELOG.md`
- `docs/开发进度.md`

## 验证

- `npm run typecheck`
- `npm run build`

## 结果说明

- 新默认主题变得更现代、清爽、年轻，不再有暖土色带来的偏旧观感。
- 界面仍然保持浅色、简约、不毛玻璃、不黑色，也避免回到之前那种偏青色的科技风。
