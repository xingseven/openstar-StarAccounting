# 2026-04-01 根目录 npm 脚本恢复旧版 web 入口

## 调整目标

- 让仓库根目录下的 `npm run dev`、`npm run build` 等命令重新指向旧版 `web/` 项目。

## 本次修改

- 将根目录 `package.json` 中的 `dev:web` 改为先进入 `web/` 再启动 `next dev`。
- 将 `build`、`start`、`lint`、`typecheck`、`test:e2e` 与 `test:e2e:ui` 同步改为代理到 `web/`。
- `dev:legacy` 统一复用修正后的 `dev:web`。

## 结果

- 现在从项目根目录执行 npm 命令，会重新落到旧版 `web/`，不会再误跑到仓库根目录自身的 Next 环境。

## 验证记录

- 根目录执行 `npm run typecheck` 已恢复为调用 `web/` 校验链路。
