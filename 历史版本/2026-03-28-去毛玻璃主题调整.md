# 2026-03-28 去毛玻璃主题调整

## 调整原因

- 当前主题虽然完成了模块分色，但半透明表面、模糊浮层和玻璃感容器仍然偏强。
- 根据体验反馈，界面需要更稳、更实，而不是继续保持悬浮和发亮的质感。

## 本次修改

- 默认主题和主题注册表统一改成实色壳层、实色表面和更明确的边框体系。
- 共享主题原语去掉模块色渐变叠层，改为纯色背景。
- 移除消费页桌面筛选浮层、Hero 半透明信息卡，以及仪表盘等页的主要模糊光斑。

## 结果

- 模块识别度继续保留。
- 毛玻璃感明显下降。
- 页面整体更像专业工作台，而不是悬浮玻璃界面。

## 验证记录

- `npm run typecheck` 通过。
- `npm run lint -- ...` 通过。
- `npm run build` 通过。
