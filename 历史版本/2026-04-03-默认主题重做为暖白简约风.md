# 2026-04-03 默认主题重做为暖白简约风

## 本次变更

- 将默认主题从偏冷蓝的科技风切换为暖白、米灰、蓝灰点缀的简约方案。
- 重做共享主题原语，收紧圆角、阴影和容器质感，去掉装饰性毛玻璃语言。
- 同步调整首页总览 Hero、收支卡、图表配色、最近交易区以及侧边栏、顶部栏、移动底部导航。

- `npm run typecheck`
- `npm run build`

## 结果说明

- 整体界面更简约，但通过暖白底、纸感层次和低饱和强调色保持了足够的精致感。
- 首页模块依旧清晰，但不再依赖青蓝色和玻璃感营造“现代感”，更适合长期使用。
